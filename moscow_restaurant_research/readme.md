# Исследование рынка общественного питания в Москве
## Данные:
  - Id заведения
  - название
  - сетевое/не сетевое
  - тип заведения
  - адрес
  - количество посадочных мест
## Задача:
  Анализ заведений Москвы, с целью определения наиболее выгодной бизнес модели.
## Используемые библиотеки:
  - pandas
  - numpy
  - seaborn
  - matplotlib
  - plotly
  - requests
  - json
