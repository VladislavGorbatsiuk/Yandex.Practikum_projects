# Маркетиноговая аналитика Яндекс.Афиши
## Данные:
  - visits:
    - тип устройства
    - начало сессии
    - конец сессии
    - id клиента
    - id источника
  - orders:
    - время покупки
    - выручка
    - id клиента
  - costs:
    - id источника 
    - дата
    - сумма потраченная в это дату
## Задача:
  На основе данных Яндекс.Афиши выделить невыгодные источники трафика.
## Используемые библиотеки:
  - pandas
  - numpy
  - seaborn
  - matplotlib
  - datetime
